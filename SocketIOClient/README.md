# Socket.IO Client

## Použití
```
usage: java -jar socket-io-client.jar [-h] [-H <hostname>] [-i <first-id>] [-n <n>] [-p <port>]

DESCRIPTION
 -h,--help                  display help
 -H,--host <hostname>       hostname (default: http://localhost)
 -i,--first-id <first-id>   id of first client (default: 1)
 -n <n>                     number of running clients (default: 1)
 -p,--port <port>           port (default: 3000)
```
